      Automatic code-splitting
      

        The Angular CLI detects components inside @defer and automatically moves them
        into separate chunks at build time. No extra configuration needed — the same mechanism used by
        loadComponent in routes, but applicable to any part of the template.
        The @placeholder block renders immediately; the main content is downloaded
        and rendered only when the trigger fires.
      

    

  


  

    
⚠

    

      When not to use @defer
      

        Components above the fold, critical for LCP (Largest Contentful Paint), or that require
        immediate SSR hydration should not be deferred.
        Reserve @defer for secondary content: charts, rich-text editors,
        data grids, inactive tabs, and anything below the initial visible screen.
